### Dependencies

- SDL2
- SDL2 Image

On **Ubuntu/Debian**:

```
sudo apt update
sudo apt install libsdl2-dev
sudo apt install libsdl2-image-dev
```

On **Mac** (I haven't actually tested compiling it on Mac, the gcc flags might be wrong):

```
brew install sdl2
brew install sdl2_image
```
